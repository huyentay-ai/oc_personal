# Dashboard thu chi

Dashboard này được tạo tự động từ file:

- `memory/notes/so_thu_chi.md`

## Cách cập nhật

Mỗi khi sửa sổ thu chi, chạy:

```bash
python3 dashboard/scripts/build_thu_chi_dashboard.py
```

Script sẽ cập nhật:

- `dashboard/data/thu_chi_data.json`
- `dashboard/thu_chi_dashboard.html`

## Cách xem

Mở file sau bằng trình duyệt:

- `dashboard/thu_chi_dashboard.html`
